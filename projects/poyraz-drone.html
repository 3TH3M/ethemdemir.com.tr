Poyraz Drone

Autonomous Border Security System

Security & Surveillance 2022
Finalist 2022 International Free Mission UAV Competition
12 Months Development Time
8 Members Team Size

Project Overview

The Poyraz Drone is an advanced autonomous border security system designed to enhance surveillance and monitoring capabilities. This innovative solution combines cutting-edge computer vision, autonomous navigation, and real-time data processing to provide comprehensive border security coverage.

Key Features

Autonomous Patrol

AI-powered autonomous navigation system for predefined patrol routes with obstacle avoidance capabilities.

Object Detection

Advanced computer vision algorithms for real-time object detection and classification.

Facial Recognition

Integrated facial recognition system for identifying individuals in the surveillance area.

Real-time Data Transmission

Secure and reliable data transmission system for live video feed and telemetry data.

Technical Specifications

Flight Performance

  • Maximum Flight Time: 45 minutes
  • Range: 15km
  • Maximum Speed: 60km/h
  • Operating Altitude: 100-500m

Sensors & Cameras

  • 4K RGB Camera
  • Thermal Imaging Camera
  • LIDAR for Obstacle Detection
  • GPS & IMU Integration

AI Capabilities

  • YOLOv5 Object Detection
  • Deep Learning-based Facial Recognition
  • Autonomous Path Planning
  • Real-time Data Processing
